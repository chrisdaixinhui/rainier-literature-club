## 1. 项目概览

### 1.1 背景

雨山前书会（Rainier Literature Society）是根植于西雅图的中文读书社群，由一群热爱中文文学的理想主义者组成。社群致力于在英语母语者的环境中推广中文阅读，降低阅读门槛，以书会友。

### 1.2 目标

| 目标类型 | 具体目标 |
|---------|---------|
| 品牌目标 | 建立可信、有文学气质的线上门面，统一社群形象 |
| 增长目标 | 扩大社群曝光，吸引西雅图华人社区新成员 |
| 运营目标 | 简化活动发布、订阅通知、周边销售等日常运营流程 |
| 社区目标 | 打造华人知识共享平台，促进线下连接 |

### 1.3 目标用户

**主要用户**：居住在西雅图及周边地区的中文使用者，年龄 20–45 岁，热爱阅读或对文学感兴趣，希望在英语环境中保持母语连接。

**次要用户**：对中文文化感兴趣的非华裔人士、远程关注社群动态的海外华人。

### 1.4 核心价值主张

> 在英语世界里，给中文热爱一个栖居之所。

## 2. 设计原则

| 原则 | 说明 |
|------|------|
| **诗意克制** | 保持文学气质，避免过度装饰，留白即内容 |
| **双语并重** | 中英文同时显示于页面内，各自成段，非翻译关系，无需切换按钮 |
| **现代报刊气质** | F型布局为主（主页）、Z型落地页（活动推广），版式参考 literal.club / amomentbooks.com |
| **温暖极简色系** | 品牌色：苍林深绿 `#2E463D`，背景：纸质暖白 `#FAF8F5`，克制圆角（4px），无高饱和度色块 |
| **内容优先** | 海报、照片、文字是主角，UI 是容器 |
| **低门槛运营** | 内容更新无需改代码，编辑团队可独立维护（Phase 2 接入 Notion） |
| **移动优先** | 用户主要通过手机访问，响应式布局从移动端开始设计 |

## 4. 页面详细设计

### 4.1 全局导航

#### 布局（已实现）

```
┌──────────────────────────────────────────────────────────────────┐
│                    首页     关于我们   活动板块   支持我们           │
│  雨山前 Rainier Lit ────────────────────────────────  [订阅·SUBSCRIBE] │
│                    HOME   ABOUT US   EVENTS   SUPPORT US         │
└──────────────────────────────────────────────────────────────────┘
```

- **Logo**：衬线体"雨山前" + 右侧斜体小字 "Rainier Lit"
- **导航项（4项，中英双语竖排）**：
  - 中文在上（衬线体 14px，深墨色）
  - 英文在下（无衬线体 8px，全大写，字距 0.22em，sage 绿色）
  - 激活状态：中英文均切换为苔绿 `#2E463D` + 加粗
- **订阅按钮**：苔绿实底色 CTA，点击弹出邮箱注册 Modal
- **语言切换**：不设切换按钮，双语内容同时显示
- **移动端**：hamburger 菜单，展开为全屏遮罩式导航（双语大字）
- **滚动行为**：`sticky top-0 z-50`，始终吸顶可见（不再随滚动隐藏）
- **背景**：`rgba(250,248,245,0.85)` + `backdrop-blur(14px)` 毛玻璃，下方极淡分割线
- **全局过渡**：`transition-all duration-500 ease-in-out`

### 4.2 关于我们 About Us

#### Section A：Hero 全屏落幅

**核心交互**：页面进入时触发逐行诗句 fade-in 动画，配合持续的背景雨效。

**背景**：
- 深色半透明图层覆盖于山林意境背景图之上
- 叠加 Canvas 粒子雨动效：细密斜线，透明度 15–25%，速度缓慢，方向左斜 15°
- 背景图待定（建议：西雅图山景 / 雨中林木 / 抽象水墨）

**文字内容**（逐行出现，每行间隔 0.8s，字体：Noto Serif SC）：

```
如果脚步无法抵达
那就从书里开始出发。

如果思绪飘向远方
文字便凝炼成翅膀。

我们邀你
栖息雨山前的苍林，成为一棵阅世的树，
晃动枝桠——
周围尽是沙沙回响。
```

**文字动画完成后出现**：
- 三个小 badge 徽章，从左向右依次 fade-in：
  - `📍 坐标西雅图`
  - `💬 在英语世界里坚持母语热爱`
  - `🌧️ 理想主义者的世界`
- CTA 按钮：「加入我们 Join Us →」（点击滚动至订阅 Section 或弹出 Modal）

**尺寸**：全屏 100vh，垂直居中

  {

## 5. 视觉设计规范

### 5.1 色彩系统

| 名称 | 色值 | 用途 |
|------|------|------|
| 冷白 Off-White | `#FAFAF8` | 页面主背景 |
| 深墨 Ink | `#1A1A1A` | 主要文字 |
| 雨雾灰 Mist | `#8EA9B0` | 次要文字、分割线、装饰 |
| 苔绿 Moss | `#5C7A6B` | Accent、按钮、Tag 高亮、支持者票边框 |
| 便签米黄 Note | `#F5F0E8` | Sentence of the Day 背景 |
| 错误红 | `#C0392B` | 表单错误提示 |

### 5.2 字体系统

| 用途 | 字体 | 字重 |
|------|------|------|
| 中文正文 / 标题 | Noto Serif SC | 400 / 700 |
| 英文标题 | Cormorant Garamond | 400 Italic / 600 |
| UI 小字 / 按钮 | Inter | 400 / 500 |
| 诗句 / 引用 | Noto Serif SC Italic | 400 |

### 5.3 圆角与间距

- 卡片圆角：`12px`
- 按钮圆角：`6px`（方正感，非全圆）
- 基础间距单位：`8px`
- 内容最大宽度：`1200px`，页面两侧 padding `24px`（移动端 `16px`）

### 5.4 图片风格

- 优先使用自然光、非摆拍的活动现场照
- 社群照片可加轻微暗角处理（vignette），增加胶片质感
- 海报图直接使用原始设计稿，不做额外滤镜处理

## 6. 交互与动效规范

### 6.1 Hero 雨效动画（已实现）

- **实现方式**：原生 Canvas API（`HeroRain.tsx`）
- **实际粒子参数**：
  - 数量：60 条
  - 速度：0.35–0.80 px/frame（缓慢飘落，西雅图毛毛雨氛围）
  - 长度：8–20px
  - 透明度：0.08–0.12（极克制，不遮挡文字）
  - 倾斜角：约 14°（从左上向右下，`x - 2.5` 偏移）
  - 颜色：`rgba(143,164,153,opacity)`（sage 绿）
- 性能：检测到 `prefers-reduced-motion: reduce` 则停止动画
- Hero 背景：深色苍林渐变（`linear-gradient(170deg, #1a2820, #243830, #1a2e24)`）

### 6.2 诗句逐行出现动画

- 每行文字：`opacity: 0 → 1`，`transform: translateY(12px) → 0`
- 时长：`600ms ease-out`
- 行间延迟：`800ms`
- 全部文字出现后，badge 和 CTA 按钮再 fade-in（延迟 `500ms`）

### 6.3 全局交互规范

| 元素 | 交互 |
|------|------|
| 按钮 hover | 背景色加深 10%，轻微上移 `translateY(-1px)` |
| 卡片 hover | `box-shadow` 加深，`scale(1.015)` |
| 链接 hover | 下划线从左向右展开 |
| 页面切换 | 路由跳转时淡入淡出（`opacity` 过渡 `200ms`） |
| Tab 切换 | 内容区 `opacity` 过渡 `150ms` |

### 6.4 加载状态

- 骨架屏（Skeleton）用于活动列表和票务列表
- 图片懒加载（Intersection Observer），加载前显示模糊占位图
